# 唐泉个人设计作品站 - 部署指南

## GitHub Pages 部署步骤

### 1. 准备工作
- 将本仓库推送到您的GitHub账户
- 确保所有文件已在 `main` 分支

### 2. 启用GitHub Pages
1. 进入仓库的 **Settings** 选项卡
2. 在左侧菜单找到 **Pages**
3. 在 **Source** 部分选择：
   - Branch: `main`
   - Folder: `/` (根目录)
4. 点击 **Save**
