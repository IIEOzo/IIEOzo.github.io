# 📁 Webd-GitHub

纯前端文件共享工具，无需服务器，可直接部署到 GitHub Pages。

## ✨ 特性

- 🚀 **无需服务器**：完全在浏览器中运行
- 📱 **PWA 支持**：可安装为桌面/手机应用
- 💾 **本地存储**：文件存储在浏览器中，保护隐私
- 🔗 **分享链接**：生成可分享的文件链接
- 🎨 **响应式设计**：适配手机、平板、桌面
- 🔒 **隐私保护**：所有数据存储在本地

## 🚀 快速部署

### 方法一：GitHub Pages（推荐）

1. Fork 或克隆此仓库
2. 前往仓库 Settings → Pages
3. 选择 `main` 分支和 `/root` 目录
4. 点击 Save，等待部署完成

### 方法二：直接使用

直接访问：[https://yourusername.github.io/webd-github/](https://yourusername.github.io/webd-github/)
